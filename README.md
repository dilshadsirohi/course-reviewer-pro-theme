# Course Reviewer Pro - WordPress Theme

A professional WordPress theme designed for course review websites.

## Features
- Custom post type for course reviews with detailed meta fields
- Star rating system with visual rating bars
- Course comparison table template
- Platform and category taxonomy filtering
- Responsive design (mobile, tablet, desktop)
- SEO schema markup for reviews
- WordPress Customizer integration
- Widget areas for sidebar and footer
- Touch-optimized mobile experience
- Safe area support for notched phones

## Installation
1. Download or clone this repository
2. Zip the theme files
3. Go to WordPress Admin > Appearance > Themes > Add New > Upload Theme
4. Upload the ZIP file and activate
5. Go to Settings > Permalinks and click Save to register custom URLs

## Custom Post Type
- **Course Reviews** - Add reviews with ratings, pros/cons, pricing, and affiliate links

## Taxonomies
- **Platforms** - Organize reviews by platform (Udemy, Coursera, etc.)
- **Review Categories** - Categorize reviews by topic

## Requirements
- WordPress 5.0+
- PHP 7.4+
